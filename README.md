# znapper

`znapper` is a code-snapshot extension for the [Zed](https://zed.dev/) text
editor. It is inspired by the VSCode extension
[CodeSnap](https://github.com/kufii/CodeSnap).

## Planned features

- Generate an image of code.
- Match with *your* Zed theme.
- Extensive configuration, e.g.:
  - background options
  - window options
  - window decorations
  - display file properties
